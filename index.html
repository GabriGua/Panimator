Key Shortcuts

B === Pencil
E === Eraser
F === Fill Tool
S === Save Current Color
CTRL === Add Frames
. === Reuse Last Frame
Arrows Left / Right === Navigate Through the Timeline
Arrows Up / Down === Change Pixel Size of Selected Tool
Z === Undo
X === Redo
Space === Play / Pause Animation
C === Clear
Enter === Export
